Addon Custom Fee in cart WC permit administrators to added an extra fee to the Woocommerce cart. You can enter your own custom amount and label for the fee to be added to the cart. You can also add tax and change the tax class if desired. It also has the option to apply a custom fee only if the cart total exceeds the minimum amount set.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `nic-addon-custom-fee-cart.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on `WooCommerce` > `Settings` > `Extra Fee` and add your settings
